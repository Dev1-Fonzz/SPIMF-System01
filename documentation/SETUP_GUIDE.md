
# Panduan Penyediaan SPIMF

1.  **Salin Kod**: Salin kandungan dari folder `appscript/` di repo ini.
2.  **Buka Apps Script**: Pergi ke [script.google.com](https://script.google.com) dan buat projek baru.
3.  **Tampal Kod**: Buat dua fail (`.gs` dan `.html`) dan tampal kod yang disalin.
4.  **Deploy**: Pilih `Deploy` > `New deployment`, pilih jenis `Web app`, dan setel akses kepada `Anyone`.
5.  **Sambung ke Google Sheet**: Pastikan ID Google Sheet anda betul dalam fungsi `initialize()`.
